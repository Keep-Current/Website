Keep-Current is an educational project with the purpose to help and promote machine learning and biotech research.

What is the Keep-Current Project?

Keep Current Project is an open-source project that aims to assist students, researchers and courious to stay current with their profession, by filtering freshly released academic papers. It uses natural language processing (NLP), machine learning and recommendation system (RecSys) technics to adapt to the topics you wish to stay current with. We currently support Quantitive Biology and Machine Learning articles.

  • Get Started

Machine Learning Seminar

We meet monthly to dive deeper into machine learning topics. Every meeting, a different topic is chosen. The participants are expected to read it in advance and during the meetup, we discuss and enrich each other with new and personal views, as well as teaching each other.

  • To our Meetup

Applied Machine Learning Course

Learn applied machine learning using Fast.AI online MOOCs in a group with us. After watching the videos at home, you get a chance to explain to the rest of the group, what you have undestood and what you're still missing. We help each other with the machine learning projects, and dive deeper with the theory behind AI.
